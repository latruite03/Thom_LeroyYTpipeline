# Thom_LeroyYTpipeline

Pipeline minimal et répétable pour produire du contenu **YouTube + X** (persona : Thom Leroy — “tech utile, rythme adulte”).

## Arborescence

- `youtube/`
  - `video-01/` … un dossier par vidéo
  - Chaque dossier contient :
    - `script.md` (texte exact intro/outro + bullets)
    - `run-of-show.md` (timeline minute par minute)
    - `screen-checklist.md` (setup écran, onglets, zoom, notifications)
    - `assets/` (slides/thumbnails si besoin)
    - `metadata.md` (titre, description, chapitres, commentaire épinglé)

- `x/`
  - `posts/` (brouillons)
  - `templates/` (formats réutilisables)

- `assets/`
  - `brand/` (bannière, avatar, styles)
  - `slides/` (modèles)

## Règle d’or

Un seul objectif : **t’asseoir, lancer l’enregistrement, et tourner** — sans passer 2h à préparer.
